

# **Home Automation Project**

Control your home appliances using voice commands and a mobile web interface. This project integrates IoT with a smart home system for ease and convenience.

---

## **Features**
- Voice-controlled device automation.
- Mobile-friendly web interface for managing devices.
- Real-time status updates of connected devices.
- Integration with ESP8266 for wireless communication.

---

## **Technologies Used**
- **Hardware**: ESP8266, Relay Module.
- **Frontend**: HTML, CSS, JavaScript.
- **Backend**: Python (Flask).
- **Voice Control**: SpeechRecognition Library.
- **IoT Communication**: HTTP Protocol.

---


## **Future Improvements**
- Add support for more devices.
- Implement scheduling features.
- Cloud integration for remote access.
